# Mineradio Android\r\n\r\n> 把 Mineradio 桌面版音乐播放器搬到 Android 上！\r\n> 原项目：https://github.com/XxHuberrr/Mineradio\r\n\r\nMineradio 是一款沉浸式音乐播放器，融合天气电台、歌词舞台、粒子视觉和 3D 歌单架。\r\n本仓库将其 Electron 桌面版移植为 Android APK，保留完整的 UI 和动画体验。\r\n\r\n---\r\n\r\n## 下载 APK\r\n\r\n### 方法一：直接下载（推荐）\r\n\r\n点此下载最新 APK：\r\nhttps://github.com/bolaosi/Mineradio-Android/releases/latest\r\n\r\n安装要求：\r\n- Android 8.0+（API 24+）\r\n- 建议 4GB 内存以上\r\n- 需要开启允许安装未知来源应用\r\n\r\n### 方法二：自行构建（GitHub Actions 云端构建，免费）\r\n\r\n如果你有 GitHub 账号，可以在云端免费构建，无需本地安装任何开发工具。\r\n\r\n步骤：\r\n1. Fork 本仓库到你自己的 GitHub\r\n2. 进入你仓库的 Actions 标签\r\n3. 左边栏点击 Build Mineradio APK\r\n4. 点击右边 Run workflow -> 再点 Run workflow\r\n5. 等待 5-10 分钟，构建完成\r\n6. 点进运行记录，在 Artifacts 下载 Mineradio-Android-APK\r\n\r\n---\r\n\r\n## 核心功能\r\n\r\n| 功能 | 支持 | 说明 |\r\n|------|------|------|\r\n| 网易云音乐搜索/播放 | 支持 | 内嵌 NeteaseCloudMusicApi |\r\n| QQ 音乐搜索/播放 | 支持 | 同上 |\r\n| 3D 粒子视觉特效 | 支持 | 基于 Three.js |\r\n| 滚动歌词 | 支持 | 完整歌词舞台 |\r\n| 天气电台 | 支持 | 根据天气推荐 |\r\n| 均衡器 | 支持 | 内置 DSP 效果 |\r\n| 本地音乐播放 | 开发中 | 待完成 |\r\n| 通知栏控制 | 计划中 | 待完成 |\r\n\r\n---\r\n\r\n## 技术架构\r\n\r\nWebView (Android) -> index.html + Three.js + GSAP\r\nNode.js (nodejs-mobile) -> server.js + NeteaseCloudMusicApi\r\nElectron API 存根 -> 桌面功能兼容层\r\n\r\n---\r\n\r\n## 项目结构\r\n\r\nmineradio-android/\r\n  .github/workflows/build-apk.yml   自动构建\r\n  app/build.gradle                   构建配置\r\n  app/src/main/AndroidManifest.xml\r\n  app/src/main/assets/index.html     Web 前端\r\n  app/src/main/assets/vendor/        第三方库\r\n  app/src/main/assets/nodejs-project/ 后端\r\n  app/src/main/java/.../MainActivity.java\r\n  app/src/main/java/.../NodeService.java\r\n\r\n---\r\n\r\n## 本地开发\r\n\r\n需要先安装 Android Studio、JDK 17+、Node.js 18+\r\n\r\n  cd app/src/main/assets/nodejs-project\r\n  npm install\r\n\r\n然后用 Android Studio 打开项目根目录，Build -> Build APK(s)\r\n\r\n---\r\n\r\n## 已知问题\r\n\r\n- 桌面歌词：Android 上不可用（已禁用）\r\n- 壁纸模式：Android 上不可用（已禁用）\r\n- 全局快捷键：Android 上不可用\r\n- Google Fonts：首次加载需联网\r\n- Node.js 后端首次启动会慢一些\r\n\r\n---\r\n\r\n## 许可\r\n\r\n本项目遵循原项目 XxHuberrr/Mineradio 的许可协议。\r\n
